# Тесты для лабораторной работы №4. Задание №2

## Входные данные
Две строки.

## Выходные данные
Слова, встречающиеся в обоих строках один раз.

## Позитивные тесты
- 01 - Обычный тест.
- 02 - Различные комбинации разделителей между словами, одинаковые слова в одной и той же строке.
- 03 - Различные комбинации разделителей, слова максимального размера.
- 04 - Повторяющиеся несколько раз слова в одной строке.
- 05 - Первая строка пустая.
- 06 - Вторая строка пустая.

## Негативные тесты
- 01 - Нет ввода.
- 02 - Первая строка больше допустимого размера.
- 03 - Вторая строка больше допустимого размера.
- 04 - Нет второй строки.
- 05 - Слово больше допустимого размера в первой строке.
- 06 - Слово больше допустимого размера во второй строке.
- 07 - Все слова повторяются.
